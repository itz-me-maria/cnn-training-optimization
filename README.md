# CNN Training Optimization and Efficient Architectures

## Overview

This project explores techniques for improving deep learning models for image classification, focusing on training monitoring, hyperparameter tuning, and efficient architectures.

---

## Features

* Custom Trainer class for training and evaluation
* Hyperparameter tuning
* Efficient CNN design (depthwise separable convolutions, SE blocks)
* MLP-Mixer implementation
* Training on CIFAR-10 dataset

---

## Technologies

* Python
* PyTorch
* Matplotlib
* Weights & Biases (wandb)

---

## Results

* Training loss: ~0.87
* Validation loss: ~0.82

The model shows stable training and good generalization.
